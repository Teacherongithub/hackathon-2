# Research: Research paper on Module 1: The Robotic Nervous System (ROS 2)

## Research Tasks

### 1. Best Practices for `rclpy` and ROS 2

**Task**: Find best practices for `rclpy` and ROS 2 in the context of humanoid robotics.
**Findings**:
- **Decision**: Adhere to the official ROS 2 documentation and the `rclpy` API guidelines. Focus on clear, well-documented examples that demonstrate the core concepts of nodes, topics, services, and parameters.
- **Rationale**: The official documentation is the most reliable source of information and ensures that the examples are up-to-date and follow best practices.
- **Alternatives considered**: None, as the official documentation is the authoritative source.

### 2. Docusaurus for Technical Documentation

**Task**: Research the use of Docusaurus for publishing technical documentation.
**Findings**:
- **Decision**: Use Docusaurus to build and publish the research paper as a static website.
- **Rationale**: Docusaurus is a popular, open-source static site generator that is well-suited for documentation projects. It provides a clean, readable layout, and supports Markdown, which is the required format for the paper. It also allows for easy hosting on GitHub Pages.
- **Alternatives considered**:
    - **MkDocs**: Another popular static site generator, but Docusaurus has better support for React components and a more modern look and feel.
    - **Jekyll**: A more general-purpose static site generator, but Docusaurus is more focused on documentation and provides more out-of-the-box features for this use case.

### 3. Finding Peer-Reviewed Sources

**Task**: Identify a strategy for finding at least 6 peer-reviewed or authoritative sources on ROS 2 and humanoid robotics published within the last 10 years.
**Findings**:
- **Decision**: Use academic search engines such as Google Scholar, IEEE Xplore, and the ACM Digital Library to find relevant papers. Also, consult the official ROS 2 documentation and other authoritative sources in the robotics community.
- **Rationale**: These search engines provide access to a wide range of peer-reviewed literature. The official ROS 2 documentation is an authoritative source.
- **Alternatives considered**: None, as these are the standard methods for academic research.
